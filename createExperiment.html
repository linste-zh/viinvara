Create Remote Experiment

This page allows you to create a full settings file that can be sent to other people, who can in turn upload it under https://linste-zh.github.io/viinvara/loadExperiment.html.
How does this work?

  1. Fill out the variable name (required).
  2. Fill out the participant name (optional, they can change this themselves again).
  3. Add a message to be shown to the participant (optional).
  4. Adjust the settings (including the website theme) to your liking.
  5. Download the file.
  6. Send it and your video file to the participant and direct them to https://linste-zh.github.io/viinvara/loadExperiment.html
  7. They will only be able to fill out their name/participant ID and start the experiment.
